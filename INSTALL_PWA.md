# Установка Poytakht CRM на главный экран

## Android (Chrome)

1. Открой сайт в Chrome: `https://web-production-f404d3.up.railway.app`
2. Войди в систему
3. Нажми меню ⋮ (три точки) → **«Добавить на главный экран»**
4. Нажми **«Добавить»**
5. Иконка Poytakht CRM появится на главном экране

## iPhone / iPad (Safari)

1. Открой сайт в Safari: `https://web-production-f404d3.up.railway.app`
2. Войди в систему
3. Нажми кнопку **«Поделиться»** (квадрат со стрелкой вверх)
4. Прокрути вниз → **«На экран «Домой»»**
5. Нажми **«Добавить»**
6. Иконка Poytakht CRM появится на рабочем столе

## Требования

- Сайт должен работать по HTTPS (уже настроено на Railway)
- Chrome 76+ или Safari 11.1+ на iOS
- Браузер Firefox не поддерживает PWA на Android

## Примечания по безопасности

- Service Worker кеширует **только** статические файлы (CSS, JS, иконки)
- Личные данные (клиенты, платежи, финансы, дашборд) **никогда не кешируются**
- После выхода из системы все приватные страницы недоступны
