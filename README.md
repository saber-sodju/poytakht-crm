# Poytakht Inshoot — CRM/ERP Система

Полноценная CRM система для управления строительным бизнесом: продажи квартир, клиенты, платежи, расходы, строительство.

## Быстрый старт

### 1. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 2. PostgreSQL
Создайте БД:
```sql
CREATE DATABASE poytakht_crm;
```

Или отредактируйте `.env` для своих настроек.

### 3. Миграции и демо-данные
```bash
python manage.py migrate
python manage.py seed_data
```

### 4. Запуск
```bash
python manage.py runserver
```

Откройте: **http://localhost:8000**

## PWA — Установка на телефон

Сайт можно добавить на главный экран как нативное приложение.

**Android (Chrome):**
1. Открыть сайт → меню ⋮ → **«Добавить на главный экран»**

**iPhone / iPad (Safari):**
1. Открыть сайт → кнопка «Поделиться» → **«На экран Домой»**

Иконка появится на рабочем столе, приложение откроется в standalone-режиме (без адресной строки).

> Безопасность: Service Worker кэширует только CSS/JS/иконки. Личные данные CRM (клиенты, платежи, дашборд) никогда не кэшируются.

## Архитектура

```
apps/
├── accounts/     # Пользователи, роли, уведомления
├── complex/      # ЖК → Блоки → Этажи → Квартиры
├── clients/      # Клиенты, лиды (CRM)
├── sales/        # Продажи, бронирования
├── payments/     # Платежи, рассрочки, долги, квитанции PDF
├── expenses/     # Расходы компании
├── workers/      # Рабочие, бригады, посещаемость, зарплата
├── materials/    # Склад материалов, поставщики, движение
├── audit/        # Журнал действий
└── dashboard/    # Дашборд, отчёты
```

## Роли и доступы

- **Директор** — полный доступ, журнал действий, управление сотрудниками
- **Администратор** — управление комплексом, продажами, клиентами
- **Менеджер** — клиенты, лиды, бронирование, продажи
- **Бухгалтер** — платежи, расходы, финансовые отчёты
- **Клиент** — личный кабинет, своя квартира, график платежей
